# SHAN MIAO 19岁 生日快乐 🎂
## 🎮 玩法指南 (How to Use) — *For Miao* ✨

『 这是一个专门为你设计的、充满“猫性”和惊喜的小宇宙 🐾 』

### 🟢 第一步：唤醒它的感官
*   **开启摄像头**：进入页面后，请大方地允许摄像头权限。它是猫咪们的“眼睛”👀。
*   **音量加满**：所有的喵呜声和环境音都是程序根据你的动作**实时合成**产生的，戴上耳机体验更佳哦！🎧

### 🟡 第二步：与猫咪“隔空”对话 🖐️
伸出你的手，尝试以下手势来触发不同的“猫咪反应”：
*   🤘 **『Yeah!』** —— 触发庆典模式，满屏彩虹花瓣！
*   ✌️ **『Peace』** —— 祈求世界和平，猫咪会给你轻柔的喵呜。
*   👍 **『赞』** —— 你超级棒！得到猫咪的认可。
*   🤏 **『比心』** —— 隔空发送猫咪之吻。
*   🖐️ **『举手』** —— 大声打个招呼吧！

### 🔵 第三步：掷出随机的“灵魂思考” 🎲
点击下方那个 **[give me a random thought 🧠]** 按钮：
*   你会看到一颗正在运动的骰子。
*   **⚀ 至 ⚅** 每一面都代表一种猫咪性格：哲学的、逻辑的、甚至有点荒谬的。
*   *小惊喜*：长按骰子可以查看“猫咪说明书” 💡。

### 🌙 第四步：寻找那个“被隐藏的秘密” ✉️
在生日横幅里，藏着一个非常安静的入口...
1.  找到那个闪烁的 **[4th April]** 按钮。
2.  进入后，请**环顾四周**。
3.  跟着地上的**发光猫脚印**走下去。
4.  在那只小白猫手里，有一封写给你的信。📖
